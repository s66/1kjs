cookie( name[,value,options]|data[,options] )

Define in : BOM

操作cookie

参数
  • name
    String
    必需

    获取cookie的值

  • value
    String
    可选

    设置cookie的值，当value为null时，则删除当前的cookie

  • options
    Object|Integer
    可选

    设置cookie的其他配置或者是有效期

  • data
    Object
    必需

    以hashmap的key和value设置cookie的值

  • options配置信息
    keyvalue说明
    expirescookie有效期，默认是浏览器关闭则失效，单位是分钟
    pathcookie的路径范围，默认是根目录
    domaincookie的domain权限，默认是当前domain
