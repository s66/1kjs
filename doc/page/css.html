css( name[,value]|csstext|hash )

原型方法Define in : Style

样式获取和操作

参数
  • name
    String
    必需

    返回style.name的样式值

  • csstext
    String
    必需

    设置css，标准的csstext字符串，比如margin-left:12px;color:red;

  • hash
    Object
    必需

    设置css，Object的key作为样式名，value作为样式值

  • value
    String
    可选

    设置样式，value为值，可以使用中划线也可以采用驼峰

返回值
  • String,$1k

    返回样式值或者节点包装对象
